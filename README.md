# Memory Card Game 🃏
 🎉 Test your memory, challenge your focus, and have fun pairing cards in this interactive web-based game. 
# About the Game 📖:
The Memory Card Game is a simple yet addictive game where players flip cards to find matching pairs. Match all the pairs to win the game! 🎯 But there’s more—track your score, race against time, and enjoy smooth animations with some fun visual effects.
# Features 🚀:
- Smooth Loading Screen: A loading screen welcomes you with a polished typewriter and blinking effect on the title.
- Interactive Gameplay: Flip cards, match pairs, and track your score in real time.
- Timer: See how quickly you can complete the game!
- Restart Anytime: Replay with a new shuffled board, reset timer, and fresh animations.
- Win Animation: Celebrate with a “YOU WON!” message when all pairs are matched.
